Detailed Description

The listener interface for receiving Glympse Lite events.

Any class that implements the IListenerLite interface and registers it by calling the IGlympseLite::addListener method will be called back with various Glympse events as they occur. When a listener is no longer needed, it should be unregistered by calling the IGlympseLite::removeListener method.
